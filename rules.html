SET Game Rules

Card Properties

Each card has 4 properties: number, color, shape, and shading.

Number

Each card has 1, 2, or 3 shapes on it.

Color

Each card is colored pink, green, or purple.

Shape

Each card is in the shape of a circle, square, or triangle.

Shading

Each card is shaded as solid, striped, or empty.

What is a SET?

A SET consist of 3 cards in which each of the cards' properties, when looked at individually, must be either the same on each card or different on each card.

To say it the long way: the number of shapes on each card in the SET must be all the same or all different; the color of each card in the SET must be all the same or all different; the shape of each card in the SET must be all the same or all different; and the shading of each card in the SET must be all the same or all different.

Quick Check

To quickly check the validity of your SET, see if any of the properties have 2 of the same with only 1 different option. For example, are there 2 green cards and 1 pink card? Are there 2 solid cards but 1 striped? Any combination where this happens is automatically not a set.
