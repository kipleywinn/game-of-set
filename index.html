SET Card Game

SET Rules

Each card has 4 properties:

  • Color (pink, purple, green)
  • Number (1, 2, 3)
  • Shape (square, circle, triangle)
  • Shading (solid, striped, empty)

A set is formed by finding 3 cards so that each category is either all the same or all different across all 3 cards.

For example, you can have 3 pink cards, or one of each color, but not 2 pink and 1 green. You can have 3 cards with 2 shapes on them, or you can have 1-2-3, but not 2 with 1 shape and 1 with 3 shapes.

If you cannot find any sets on the board, use the shuffle button to re-deal from the remaining cards.
